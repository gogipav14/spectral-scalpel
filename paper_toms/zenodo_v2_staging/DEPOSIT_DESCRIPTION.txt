Reproducibility archive for the manuscript "scalpel: A multi-backend FFT-NILT primitive for spectral slab PDE solvers, with class-dependent finite-precision auto-tuning" (Gorgi Pavlov, under review at ACM Transactions on Mathematical Software, 2026).

The archive contains:

* The full scalpel Python library (Apache-2.0): a batched FFT-accelerated numerical inverse Laplace transform primitive for spectral slab PDE solvers, with 26 public top-level callables across 6 modules (api, chunked, diagnose, sweep, validate, config), running unchanged across NumPy, PyTorch, and JAX under a unified dispatcher built on top of the Python Array API, plus CuPy and Julia via reference wrappers.

* The class-dependent finite-precision parameter auto-tuner backed by two closed-form bounds (one per dispersion class, telegrapher and diffusion), with the mathematical derivation in the manuscript's Section 4 and the implementation in scalpel/core/feasibility.py.

* Four bundled physics dispersions: lossy Maxwell, preparative chromatography (both Cartesian and natively cylindrical Hankel-transform paths), thermoviscous acoustics, and elastodynamics (P/S waves), plus a fractional Caputo single-term pencil as a stress test.

* Validation infrastructure: 14 pytest files exercising the library, four matched-accuracy baselines (Yee FDTD, FTCS+L1, de Hoog QD, fixed Talbot), and an mpmath 50-digit Talbot ground-truth reference matched to 8e-4 relative error on the validation depth.

* The reproducibility pipeline: a one-command `bash paper_toms/reproducibility/reproduce.sh` driver (measured 21 s wall time end-to-end on the reference workstation), a CSV-diff verification utility, and a multi-OS x multi-Python x multi-backend GitHub Actions CI matrix.

* The full manuscript source (LaTeX + figures + CSVs) and the four pre-submission audit / referee reports under paper_toms/reviews/.

Reference workstation: Intel Core Ultra 5 225F (10-core CPU), 32 GB DDR5, NVIDIA RTX 5060 (8 GB VRAM, consumer GPU), WSL2 (Linux kernel 6.6) on Windows 11, CUDA 13.

This v2 deposit supersedes v1 (10.5281/zenodo.19834321, deposited 2026-04-27 under the SISC-era manuscript title) following the manuscript's reframe from algorithm paper to software paper for ACM TOMS.
